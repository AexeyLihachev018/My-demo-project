# Project-1 — Мини-сайт Промгидравлика

## Что это за проект

Отдельный мини-сайт группы компаний «Промгидравлика» (Ижевск), независимый от основного сайта p-gidravlika.ru.
Ориентирован на клиентов, которым нужно быстро подобрать и заказать гидроцилиндр или гидростанцию индивидуального изготовления.
Всё реализовано на чистом HTML/CSS/JS — без фреймворков и внешних зависимостей.

## Для кого

- Инженеры-конструкторы и технологи, подбирающие гидравлику под задачу
- Руководители производственных предприятий, ищущие поставщика под заказ
- Новые клиенты, впервые знакомящиеся с компанией

## Дизайн

- **Цветовая схема:** тёмный фон `#060810` / `#030b1a`, акцентный цвет — синий `#3b82f6` (index.html); оранжевый `#f97316` утверждён брифом как целевой для будущих страниц
- **Типографика:** Arial Black / Arial, sans-serif — жирная, техническая, без засечек
- **Компоненты:** карточки с `backdrop-filter: blur`, тени через акцентный цвет
- **Анимации:** IntersectionObserver scroll-reveal, pulse, ripple, bounce
- **Адаптив:** мобильное меню-оверлей, flex/grid-раскладка, breakpoint 900px
- **Иконки:** только inline SVG — нет внешних зависимостей
- **Логотип:** SVG с шестигранником и надписью ПРОМГИДРАВЛИКА / ГРУППА КОМПАНИЙ

> Примечание: `index.html` (основная страница) использует синий акцент `#3b82f6`. Бриф (brief.md) указывает оранжевый `#f97316` как целевой. Решение о перекраске — на усмотрение владельца проекта.
