# Project-1 — Мини-сайт Промгидравлика

## Что это за проект

Отдельный мини-сайт группы компаний «Промгидравлика» (Ижевск), независимый от основного сайта p-gidravlika.ru.
Ориентирован на клиентов, которым нужно быстро подобрать и заказать гидроцилиндр или гидростанцию индивидуального изготовления.

Проект состоит из двух независимых модулей:
- **Лендинг-калькулятор** — HTML-страница, деплой на Vercel
- **Telegram Mini App** — каталог гидроцилиндров с ботом, деплой на Beget VPS

## Для кого

- Инженеры-конструкторы и технологи, подбирающие гидравлику под задачу
- Руководители производственных предприятий, ищущие поставщика под заказ
- Новые клиенты, впервые знакомящиеся с компанией

## Telegram Mini App (`tg-app/`)

**Бот:** [@PromGidravlika_bot](https://t.me/PromGidravlika_bot)
**Продакшн:** https://193.168.48.98:8443 (Beget VPS)
**Фронтенд (Mini App):** https://my-demo-project-nt8u.vercel.app

### Функционал

- Каталог: 26 позиций гидроцилиндров, фильтрация по диаметру и давлению
- Поиск по артикулу, диаметру, ходу
- Калькулятор (2 шага): усилие + давление → диаметр ГОСТ → полный расчёт
- Заявка (корзина) с отправкой менеджеру через `tg.sendData`
- Страница мастера по slug: `/m/:slug`
- Онбординг, оффер-модалка, нижняя навигация (5 вкладок)

### API endpoints

| Endpoint | Файл | Описание |
|----------|------|----------|
| `POST /webhook` | `api/webhook.js` | Обработка команд и заявок бота |
| `GET /api/set-webhook` | `api/set-webhook.js` | Регистрация webhook (одноразово) |
| `POST /webhook` (platform) | `api/platform-webhook.js` | Онбординг мастеров |
| `GET /m/:slug` | `api/m/[slug].js` | HTML-страница мастера |
| `GET /api/master/:slug` | `api/master/[slug].js` | JSON данных мастера |

## Дизайн

- **Лендинг:** тёмный фон `#030b1a`, акцент синий `#3b82f6`
- **Mini App:** тема берётся из Telegram SDK (`--tg-theme-*`), акцент `#2aabee` / `#f97316`
- **Будущие страницы сайта:** оранжевый акцент `#f97316` (по брифу)
- **Типографика:** Arial Black / Arial, sans-serif
- **Компоненты:** карточки с `backdrop-filter: blur`, тени через акцентный цвет
- **Адаптив:** от 600px (цель 320px), breakpoint 900px
- **Иконки:** только inline SVG

## Технический стек

| Компонент | Технология |
|-----------|-----------|
| Фронтенд лендинга | Vanilla HTML5 / CSS3 / JS (без зависимостей) |
| Telegram Mini App | Vanilla HTML/CSS/JS + Telegram Web App SDK |
| Серверная часть | Node.js 20 (ES modules), Express |
| База данных | Supabase (PostgreSQL) |
| Очереди задач | Upstash QStash (напоминания за 24ч) |
| Процесс-менеджер | PM2 (автозапуск, авторестарт) |
| Хостинг бота | Beget VPS, Ubuntu 24.04, IP 193.168.48.98 |
| Хостинг фронта | Vercel (автодеплой при git push) |
| SSL | Самоподписанный сертификат (порт 8443) |
| Git-репозиторий | https://github.com/AexeyLihachev018/My-demo-project |
