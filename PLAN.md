## Критические исправления 🔴

- [ ] Форма "Заказать звонок" — код EmailJS уже встроен в `landing.html`, нужно настроить аккаунт:
  1. Зарегистрироваться на emailjs.com (бесплатно, 200 писем/мес)
  2. Добавить Email Service (Gmail / Яндекс / SMTP) → получить `Service ID`
  3. Создать Email Template с переменными `{{from_name}}`, `{{phone}}`, `{{page}}` → получить `Template ID`
  4. Скопировать `Public Key` из Account → API Keys
  5. Заменить три плейсхолдера в `landing.html`: `ВАШ_PUBLIC_KEY`, `ВАШ_SERVICE_ID`, `ВАШ_TEMPLATE_ID`
- [ ] Нет навигации между страницами — добавить меню с ссылками на все разделы
